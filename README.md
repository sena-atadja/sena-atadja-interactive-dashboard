# Beats Speaker Launch Dashboard

Interactive dashboard analyzing consumer insights to support the launch of a premium wireless speaker.

## Live Interactive Dashboard
**Click here to explore the interactive dashboard:**
[Launch dashboard]  (https://claude.ai/public/artifacts/f8f9788a-4758-475f-801b-2fe293c87498)

## Key Insights
- 77% of respondents are aged 18–24
- Strong demand in the $100–$200 price range
- Top features: sound quality, battery life, design

## Tools Used
- Claude AI (dashboard generation)
- React (JSX)
- Recharts (data visualization)

## Purpose
Simulates a real-world product strategy and analytics workflow.
